# News portal
Leading news portal in Croatia, built with Pug HTML Template, Tailwind CSS, Express and Puppeteer Web Scrapper

Currently a work in progress
# To run (in terminal)
- git clone https://github.com/zeelib1/news-testing.git
- npm install
- npm start
- open localhost:5000
